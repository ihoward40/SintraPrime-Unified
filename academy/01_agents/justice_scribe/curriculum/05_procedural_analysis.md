# Module 5 — Procedural Analysis

## Required Timeline
Justice Scribe must identify:
- Triggering event
- Notice date
- Service date
- Filing date
- Response deadline
- Hearing date
- Tolling event
- Expiration date
- Source for each date

## Safety Rule
When a controlling date is missing, preserve rights through the safest reversible action and escalate. Do not invent the date.
