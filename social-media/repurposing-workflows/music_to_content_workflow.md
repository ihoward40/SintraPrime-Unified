# Content Repurposing Workflow — Music to Content

**Purpose:** Turn one song, topic, or legal education point into a full content ecosystem across all platforms.

---

## The Workflow

### Step 1: Start With One Asset

Choose one:
- A song (lyrics, theme, hook)
- A legal education point (consumer right, statute, procedure)
- A personal story (legacy, family, experience)
- A product feature (CECC tier, template, scorecard)

### Step 2: Extract the Core Elements

| Element | Question to Answer | Output |
|---------|-------------------|--------|
| **Main Hook** | What grabs attention in 3 seconds? | 1 sentence |
| **Main Lesson** | What does the audience learn? | 1 paragraph |
| **Best Lyric Line** | What line could stand alone as a post? | 1 quote |
| **Visual Concept** | What would the video look like? | 1 sentence |
| **Audience Pain Point** | What problem does this solve? | 1 sentence |
| **CTA** | What should they do next? | 1 link or action |

### Step 3: Create Platform Versions

| Platform | Format | Length | Key Element |
|----------|--------|--------|-------------|
| **TikTok** | Vertical video | 15-60 sec | Hook + lesson + CTA |
| **YouTube Short** | Vertical video | 15-60 sec | Hook + lesson + CTA |
| **Instagram Reel** | Vertical video | 15-90 sec | Visual + caption + CTA |
| **Facebook Reel** | Vertical video | 15-60 sec | Hook + CTA |
| **X Post** | Text + media | 1-4 tweets | Quote or lesson |
| **X Thread** | Text series | 5-15 tweets | Deep dive |
| **YouTube Long-Form** | Horizontal video | 5-20 min | Full lesson |
| **Email** | Text | 1-3 paragraphs | Value + CTA |
| **Website Blog** | Text + media | 500-1000 words | Full article |

### Step 4: Create Three Versions

| Version | Tone | Goal | Example |
|---------|------|------|---------|
| **Educational** | Informative, clear | Teach the lesson | "Here's what FCRA § 611 actually says..." |
| **Emotional / Legacy** | Personal, resonant | Build connection | "This song is for my daughter..." |
| **Sales / CTA** | Direct, actionable | Drive conversion | "Your evidence file might be missing this..." |

### Step 5: Track Performance

| Metric | What It Tells You |
|--------|-------------------|
| Views / Plays | Reach — how many saw it |
| Likes / Saves | Resonance — how many valued it |
| Comments | Engagement — how many reacted |
| Shares | Advocacy — how many amplified it |
| CTA clicks | Conversion — how many took action |

### Step 6: Double Down

- If educational wins → make more educational content
- If emotional wins → lean into legacy storytelling
- If sales wins → optimize the CTA
- If a specific platform wins → prioritize that platform

## Example: Repurposing "Undeniable Evidence"

| Element | Output |
|---------|--------|
| **Main Hook** | "Your dispute letter got ignored. Here's why." |
| **Main Lesson** | Evidence is what moves a dispute from words to proof |
| **Best Lyric Line** | "Paper without proof is just noise" |
| **Visual Concept** | Split screen: typed letter vs. evidence folder |
| **Pain Point** | Disputes that go nowhere |
| **CTA** | Link to CECC Evidence Audit ($97) |

**Platform outputs:**
- TikTok: 60-sec hook + lesson + CTA
- YouTube Short: Same video, different caption
- X Post: "Paper without proof is just noise. — Undeniable Evidence"
- Email: "Why your dispute got ignored (and what to do instead)"
- Blog: "The 3 Questions Every Dispute Must Answer Before You Send It"
