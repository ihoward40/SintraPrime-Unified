# Trust Records — ISIAH TARIK HOWARD TRUST

**Purpose:** This folder holds all records related to the administration, governance, assets, and beneficiaries of the ISIAH TARIK HOWARD TRUST. Nothing important should live only in memory.

## Record-Keeping Rules

1. **Every document answers:**
   - What is it?
   - Who created it?
   - When was it created or received?
   - Why does it matter to the trust?
   - Where is the original stored?

2. **Folder structure:**

   | Folder | Purpose |
   |--------|---------|
   | `assets/` | Inventory of trust assets — accounts, property, policies, IP, receivables |
   | `beneficiaries/` | Beneficiary registry — names, relationships, contact info, distribution records |
   | `governance/` | Trust instrument, amendments, trustee records, resolutions, minutes |
   | `records/` | Correspondence, tax filings, annual reviews, receipts, notices |

3. **No sensitive PII in filenames.** Use initials or case numbers.

4. **Update frequency:** Every time a trust-relevant document is created or received, log it here within 24 hours.

5. **Annual review:** Every calendar year, run the governance checklist and update all records.
