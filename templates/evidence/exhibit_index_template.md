## Exhibit Legend

Each exhibit entry answers:

- **What is it?** — Document type and brief description
- **Who created/sent it?** — Originator
- **When?** — Date on the document
- **Why does it matter?** — What claim or defense does it support?
- **Where is it stored?** — File path in the evidence folder

---

## Usage Notes

- Number exhibits sequentially (Exhibit 1, 2, 3...)
- When referencing in correspondence, use format: **Exhibit 1** (Description)
- When a document is added between existing exhibits, re-number or use Exhibit 1-A format
- Keep the index updated every time a document is added or removed
