CORE METHODS
============

execute(task_id: str, command: str) → Dict
    Execute a task asynchronously
    Returns: {"status": "success|failed", "task_id": "...", "result": {...}}

learn(task_id: str, result: Dict) → Optional[str]
    Learn from task results, generate skills
    Returns: skill_id or None

collaborate(other_agent: BaseAgent, message: Dict) → Dict
    Send message to another agent
    Returns: response dictionary


COMMON PATTERNS
===============

1. Execute a Task
   ────────────────
   import json
   import asyncio
   
   async def run():
       agent = AgentRegistry.create_agent("analyst", "Bot")
       task = {"query": "example", "analysis_type": "summary"}
       result = await agent.execute("task_1", json.dumps(task))
       return result
   
   asyncio.run(run())


2. Learn from Results
   ───────────────────
   skill_id = await agent.learn("task_1", result["result"])
   if skill_id:
       print(f"Acquired skill: {skill_id}")


3. Agent Collaboration
   ────────────────────
   message = {"request_type": "share_analysis"}
   response = await agent1.collaborate(agent2, message)


4. Check Agent Status
   ──────────────────
   status = agent.get_status()
   print(f"Status: {status['status']}")
   print(f"Skills: {status['skills_count']}")
   print(f"Success rate: {status['performance']['success_rate']:.1%}")

EXAMPLE: FULL WORKFLOW
======================

import json
import asyncio
from agent_types import AgentRegistry

async def workflow():
    # Create agents
    analyst = AgentRegistry.create_agent("analyst", "Researcher")
    executor = AgentRegistry.create_agent("executor", "Coder")
    
    # Run analysis
    task1 = {
        "query": "AI trends",
        "analysis_type": "trend",
        "depth": "medium",
        "sources": ["reports"]
    }
    result1 = await analyst.execute("t1", json.dumps(task1))
    print(f"Analysis: {result1['result']['findings']}")
    
    # Learn from analysis
    skill = await analyst.learn("t1", result1["result"])
    print(f"Skill acquired: {skill}")
    
    # Executor processes results
    task2 = {
        "code": "print('Processing results')",
        "language": "python",
        "timeout": 30
    }
    result2 = await executor.execute("t2", json.dumps(task2))
    print(f"Output: {result2['result']['output']}")
    
    # Collaborate
    msg = {"request_type": "share_analysis"}
    response = await analyst.collaborate(executor, msg)
    print(f"Collaboration: {response['status']}")

asyncio.run(workflow())

TROUBLESHOOTING
===============

Agent not executing:
  ✓ Check if task JSON is valid
  ✓ Verify required fields are present
  ✓ Check agent status before executing

Skill not acquired:
  ✓ Check if task was successful (status == "success")
  ✓ Some agents don't always acquire skills
  ✓ Check agent.skills dict for acquired skills

Collaboration failing:
  ✓ Ensure both agents are created
  ✓ Check message format is correct
  ✓ Verify request_type is supported
