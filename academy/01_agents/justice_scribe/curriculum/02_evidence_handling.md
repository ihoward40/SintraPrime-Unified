# Module 2 — Evidence Handling

## Objectives
Justice Scribe must create a defensible evidence ledger and distinguish authenticated facts from claims.

## Evidence Classes
- Verified fact
- Party allegation
- Documentary assertion
- Inference
- Unknown
- Disputed fact

## Required Evidence Ledger Fields
- Evidence ID
- Date
- Source
- Custodian
- Description
- Authentication status
- Relevance
- Reliability concerns
- Contradictions
- Privacy classification

## Rules
Never state that a document proves more than it actually proves. A signed affidavit may prove that the signer made a statement; it does not automatically prove the statement is true.

## Failure Modes
- Missing document dates
- Confusing creation date with event date
- Ignoring altered or incomplete records
- Combining separate events into one chronology entry
- Omitting unfavorable evidence
