# Sample Scorecard — Consumer Evidence Command Center

**Purpose:** Example showing how the 7-category /70 scorecard works.
**Note:** This is a sample only. Actual scores depend on the specific evidence provided.

---

## Scoring Framework

Each category scored /10 based on:
- **Severity** of the apparent issue (weighted by evidence confidence)
- **Completeness** of the evidence packet for that category
- **Actionability** — whether a clear next step exists

**Scale:** 0-3 = 🔴 Critical gap · 4-6 = ⚠️ Needs work · 7-8 = ✅ Manageable · 9-10 = 🟢 Strong

## What This Score Means

**Score: 21/70 — Critical Gaps**

This score means the evidence file is not ready for any dispute, demand, or filing. The priority is evidence collection, not action.

**Recommended next step:** Complete the Evidence Audit ($97) to get a detailed gap analysis and priority action list.

## How Scores Improve

| Action | Score Improvement |
|--------|-----------------|
| Obtain full credit reports | +5-10 points |
| Locate original contract | +5-8 points |
| Find repossession/sale notices | +8-12 points |
| Download CFPB case file | +3-5 points |
| Create complete timeline | +3-5 points |

**Target score before any action:** 50/70
