DEPLOYMENT & USAGE
================================================================================

Installation:
  1. Copy predictive/ directory to your project
  2. Install dependencies: pip install scikit-learn pandas numpy fastapi pydantic
  3. Import modules: from predictive import CaseOutcomePredictor, JudgeAnalyzer, ...

Basic Usage Example:
  
  from predictive.outcome_predictor import CaseOutcomePredictor, CaseFeatures
  from predictive.judge_analyzer import JudgeAnalyzer
  
  # Initialize models
  predictor = CaseOutcomePredictor()
  judge_analyzer = JudgeAnalyzer()
  
  # Create case features
  case = CaseFeatures(
      case_id='case_001',
      case_type='employment',
      court='federal',
      judge_name='Judge Sarah Mitchell',
      representation_quality_plaintiff=7.5,
      damages_requested=500000,
      # ... other features
  )
  
  # Get prediction
  prediction = predictor.predict(case)
  print(f"Win probability: {prediction.win_probability:.1%}")
  print(f"Confidence: {prediction.confidence_interval.lower:.1%} - {prediction.confidence_interval.upper:.1%}")
  print(f"Key factors: {prediction.key_factors}")
  print(f"Recommendations: {prediction.recommendations}")
