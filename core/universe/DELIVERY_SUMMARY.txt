To begin using the orchestration engine:

1. READ:     See /agent/home/universe/QUICKSTART.md (5 minutes)
2. REVIEW:   See /agent/home/universe/ARCHITECTURE.md (detailed design)
3. RUN:      Execute tests with: pytest test_core_engine.py -v
4. INTEGRATE: Use the API examples in README.md

Example Usage:
──────────────────────────────────────────────────────────────────────────

    import asyncio
    from core_engine import create_engine

    async def main():
        engine = await create_engine(max_agents=50)
        
        result = await engine.orchestrate(
            intent="Analyze and validate data",
            agent_descriptors=[
                {
                    'agent_id': 'analyzer_001',
                    'capabilities': ['analysis', 'validation'],
                    'max_concurrent_tasks': 5
                }
            ]
        )
        
        print(result)

    asyncio.run(main())
