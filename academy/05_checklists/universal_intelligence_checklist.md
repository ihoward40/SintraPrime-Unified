# Universal Intelligence Checklist

- [ ] Identify the actual question
- [ ] Identify jurisdiction and governing framework
- [ ] Separate facts, allegations, assumptions, and conclusions
- [ ] List missing information
- [ ] Locate primary authority
- [ ] Confirm authority is current
- [ ] Check exceptions and contrary authority
- [ ] Identify deadlines and irreversible steps
- [ ] Verify calculations
- [ ] Identify privacy, security, and authorization risks
- [ ] Draft the strongest opposing argument
- [ ] Assign a calibrated confidence score
- [ ] Identify required human approval
- [ ] Preserve sources and work product
- [ ] Update institutional memory
