## 🚀 Quick Start

### Windows
```powershell
git clone https://github.com/ihoward40/SintraPrime-Unified.git C:\SintraPrime-Unified
cd C:\SintraPrime-Unified
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m pytest
```

### Linux/Mac
```bash
git clone https://github.com/ihoward40/SintraPrime-Unified.git
cd SintraPrime-Unified
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m pytest
```

### MCP (Connect Claude Desktop)
Add to `~/.claude/claude_desktop_config.json`:
```json
{
  "mcpServers": {
    "sintraprime": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "cwd": "/path/to/SintraPrime-Unified"
    }
  }
}
```
