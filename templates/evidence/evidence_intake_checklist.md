## 10-Element Evidence Protocol

Per the Personal Think Tank protocol, every matter requires:

| # | Element | Status | Notes |
|---|---------|--------|-------|
| 1 | **Master Timeline** — chronological list of all known events with dates | ⚪ Not started | |
| 2 | **Party List** — all individuals and entities involved | ⚪ Not started | |
| 3 | **Document List** — every document in the evidence folder | ⚪ Not started | |
| 4 | **Issue List** — specific legal or factual issues identified | ⚪ Not started | |
| 5 | **Violation List** — specific laws, rules, or contract terms potentially violated | ⚪ Not started | |
| 6 | **Damages List** — actual damages, statutory damages, costs, fees | ⚪ Not started | |
| 7 | **Deadline List** — statutes of limitation, response deadlines, filing deadlines | ⚪ Not started | |
| 8 | **Correspondence Log** — all communications sent and received | ⚪ Not started | |
| 9 | **Exhibit Index** — numbered exhibits with descriptions | ⚪ Not started | |
| 10 | **Next-Action Checklist** — priority-ordered actions | ⚪ Not started | |
